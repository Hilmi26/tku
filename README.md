## 🔍 What is tku?

tku is a simple tool that helps you check how many tokens you use with AI services like Claude Code, Codex, and others. Tokens are units that these AI tools count when you send or receive words. Knowing your token use helps you keep control of costs and limits while working with AI models.

You do not need to understand programming to use tku. It works on Windows and is easy to set up and run. This guide will take you through every step to get tku working on your computer.
